Zeochat — Local Setup Instructions

Follow these steps to clone the repo, switch to development, configure the DB, and run the installer.

  1. Clone the repository 
    git clone https://gitlab.com/zeochat/Zeochat.git
  2. Switch to the development branch 
    cd Zeochat
git checkout development
  3. Create a MySQL database

    Create a database (example name: zeochat_db). Make sure you have a MySQL server running and a user with privileges to create/use the DB.

    -- example using mysql CLI
CREATE DATABASE zeochat_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
GRANT ALL PRIVILEGES ON zeochat_db.* TO 'zeouser'@'localhost' IDENTIFIED BY 'your_password';
FLUSH PRIVILEGES;
  4. Update environment file

    Copy .env.example to .env if it doesn't exist, then add your DB details.

    cp .env.example .env
# then edit .env and set:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=zeochat_db
DB_USERNAME=zeouser
DB_PASSWORD=your_password
  5. Run installer / setup

    Install dependencies and run the Metafox installer (the command you provided):

    composer install
# generate app key & migrate
php artisan key:generate
php artisan migrate --force
php artisan metafox:install --force

Troubleshooting highlights:

  • If you see PHP extension errors (e.g., ext-intl missing), install/enable the extension in your PHP.ini.
  • On Windows, if Git warns about line endings (LF/CRLF), consider setting git config --global core.autocrlf input.
  • Ensure correct PHP version, Composer installed, and MySQL credentials match .env.
